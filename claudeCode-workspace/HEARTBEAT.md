# HEARTBEAT

Checklist curto para verificacoes leves:

- existe task bloqueada sem dono claro?
- existe relatorio critico vencido?
- existe erro novo no supervisor?
- memoria do dia foi atualizada?
- existe worktree orfa, sem objetivo claro ou sem branch valida?
- existe worktree com mudancas locais sem task ou dono rastreavel?
- existe worktree limpa ou encerrada que ja pode entrar em limpeza?

Se nada precisar de atencao, responder `HEARTBEAT_OK`.
