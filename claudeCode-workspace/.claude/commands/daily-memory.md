Atualizar a memoria diaria do workspace para a data de hoje.

Passos:
1. Verificar se existe `memory/YYYY-MM-DD.md` para hoje (substitua pela data real)
2. Se nao existir, criar o arquivo com o cabecalho: `# YYYY-MM-DD`
3. Perguntar ao usuario o que deve ser registrado (ou usar o contexto da sessao atual)
4. Registrar fatos, decisoes, riscos e links para artefatos gerados
5. Avaliar se algum item merece ser promovido para `MEMORY.md`
